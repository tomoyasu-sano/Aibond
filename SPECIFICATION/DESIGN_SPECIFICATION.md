## 2. デザインシステム

### 2.1 カラーパレット

#### プライマリカラー

| 名前 | HEX | 用途 |
|------|-----|------|
| **Coral** | `#FF7F7F` | メインアクション、ブランドカラー |
| **Coral Light** | `#FFB3B3` | ホバー状態、背景アクセント |
| **Coral Dark** | `#E66666` | アクティブ状態 |

#### セカンダリカラー

| 名前 | HEX | 用途 |
|------|-----|------|
| **Soft Pink** | `#FFE4E8` | 背景、カード |
| **Cream** | `#FFF8F0` | ページ背景 |
| **Lavender** | `#E8E0F0` | セカンダリアクション |

#### テキストカラー

| 名前 | HEX | 用途 |
|------|-----|------|
| **Dark** | `#2D2D2D` | 見出し、重要テキスト |
| **Body** | `#4A4A4A` | 本文 |
| **Muted** | `#8A8A8A` | 補足テキスト、プレースホルダー |
| **Light** | `#FFFFFF` | 暗い背景上のテキスト |

#### 機能カラー

| 名前 | HEX | 用途 |
|------|-----|------|
| **Success** | `#7BC67B` | 完了、成功 |
| **Warning** | `#FFB347` | 警告、注意 |
| **Error** | `#FF6B6B` | エラー |
| **Info** | `#6BB3FF` | 情報 |

#### 話者識別カラー

視認性を高めるため、話者色は**線色・テキストアクセント**に使用し、背景は淡色にする。

| 名前 | 線・アクセント | 淡い背景 | 用途 |
|------|---------------|----------|------|
| **Speaker A** | `#E07070` | `#FFF5F5` | ユーザーA（自分）の発言 |
| **Speaker B** | `#5A9BD4` | `#F5F9FF` | ユーザーB（パートナー）の発言 |

**適用ルール**:
- 吹き出しの**左ボーダー（3px）**に線色を使用
- 話者名に線色を適用
- 背景は淡色のみ（濃い背景色は使わない）
- ブランドコーラルとの色被りを避けるため、Speaker Aは若干くすんだ赤系に調整

### 2.2 タイポグラフィ

#### フォントファミリー

| 言語 | フォント | フォールバック |
|------|----------|---------------|
| **日本語** | Noto Sans JP | sans-serif |
| **英語** | Inter | sans-serif |

#### フォントサイズ（汎用）

| 名前 | サイズ | 行高 | 用途 |
|------|--------|------|------|
| **Display** | 32px | 1.2 | ランディングページ見出し |
| **H1** | 28px | 1.3 | ページタイトル |
| **H2** | 24px | 1.3 | セクション見出し |
| **H3** | 20px | 1.4 | サブセクション |
| **Body Large** | 18px | 1.6 | 強調本文 |
| **Body** | 16px | 1.6 | 本文 |
| **Body Small** | 14px | 1.5 | 補足情報 |
| **Caption** | 12px | 1.4 | キャプション、タイムスタンプ |

#### フォントサイズ（トーク画面専用）

吹き出しが長文になると縦に間延びするため、トーク画面では行高を抑え、翻訳部分は小さめに設定。

| 名前 | サイズ | 行高 | 用途 |
|------|--------|------|------|
| **Talk Body** | 15px | 1.4 | 吹き出し内の発話テキスト |
| **Talk Translation** | 13px | 1.5 | 翻訳テキスト（メインより小さく） |
| **Talk Speaker** | 13px | 1.0 | 話者名 |
| **Talk Time** | 11px | 1.0 | タイムスタンプ |

**適用例**:
```
┌─────────────────────────────────────────────────┐
│ 太郎                                  10:05    │ ← 13px/11px
│ 今週末どうする？何か予定ある？                  │ ← 15px, 行高1.4
│ ────────────────────────────────────          │
│ What should we do this weekend?                │ ← 13px, 行高1.5, Muted色
│ Do you have any plans?                         │
└─────────────────────────────────────────────────┘
```

#### フォントウェイト

| 名前 | ウェイト | 用途 |
|------|---------|------|
| **Bold** | 700 | 見出し、強調 |
| **Semi Bold** | 600 | ボタン、ラベル |
| **Medium** | 500 | サブ見出し |
| **Regular** | 400 | 本文 |

### 2.3 スペーシング

| 名前 | サイズ | 用途 |
|------|--------|------|
| **xs** | 4px | 最小マージン |
| **sm** | 8px | アイコンとテキストの間 |
| **md** | 16px | 要素間の標準マージン |
| **lg** | 24px | セクション間 |
| **xl** | 32px | 大きなセクション間 |
| **2xl** | 48px | ページセクション間 |

#### 適用範囲の詳細

| コンポーネント | 内側padding | 外側margin | 備考 |
|---------------|-------------|------------|------|
| **吹き出し** | 12px | 8px（上下） | 情報密度を高めるため狭め |
| **カード** | 16px | 16px | 標準 |
| **リストアイテム** | 12px (上下) / 16px (左右) | 0 | ボーダーで分離 |
| **モーダル** | 24px | - | 読みやすさ確保 |
| **ページ全体** | 16px (左右) | - | SPでの左右余白 |
| **セクション間** | - | 24px (上) | ホーム画面等 |

#### トーク画面のスペーシング

```
┌─ 吹き出し ──────────────────────────────────────┐
│ padding: 12px                                   │
│ ┌─────────────────────────────────────────────┐│
││ 話者名           タイムスタンプ              ││ ← gap: 4px
│├─────────────────────────────────────────────┤│
││ 発話テキスト                                 ││ ← margin-top: 4px
│├─────────────────────────────────────────────┤│
││ 翻訳テキスト                                 ││ ← margin-top: 8px, 点線区切り
│└─────────────────────────────────────────────┘│
└─────────────────────────────────────────────────┘
  ↓ margin-bottom: 8px
┌─ 次の吹き出し ──────────────────────────────────┐
```

### 2.4 角丸（Border Radius）

| 名前 | サイズ | 用途 |
|------|--------|------|
| **sm** | 8px | 入力フィールド、小さいボタン |
| **md** | 12px | カード、ボタン |
| **lg** | 16px | モーダル、大きなカード |
| **xl** | 24px | 吹き出し、特殊なカード |
| **full** | 9999px | ピル型ボタン、アバター |

### 2.5 シャドウ

| 名前 | 値 | 用途 |
|------|-----|------|
| **sm** | `0 1px 2px rgba(0,0,0,0.05)` | 軽い浮き上がり |

| **lg** | `0 10px 15px rgba(0,0,0,0.1)` | モーダル、ドロップダウン |

## 5. レスポンシブ対応

### 5.1 ブレークポイント

| 名前 | 幅 | 対象デバイス |
|------|-----|-------------|
| **sm** | < 640px | スマートフォン |
| **md** | 640px - 768px | 大型スマホ、小型タブレット |
| **lg** | 768px - 1024px | タブレット |
| **xl** | > 1024px | デスクトップ |

### 5.2 レイアウト指針

```
スマートフォン (< 640px)

│ フル幅              │

│ 下部ナビゲーション  │
└─────────────────────┘

タブレット (768px - 1024px)
┌─────────────────────────────┐
│ 最大幅 768px、中央寄せ       │
│ シングルカラム               │
│ 下部 or サイドナビ           │

┌─────────────────────────────────────┐

│ サイドナビゲーション                 │
└─────────────────────────────────────┘
```

## 6. アニメーション・インタラクション

### 6.0 アニメーション方針

**基本原則**:
- **控えめ**: 派手すぎず、邪魔にならない
- **意味のある動き**: 状態変化を伝えるために使用
- **パフォーマンス優先**: 60fps維持、`transform`/`opacity`のみ使用
- **アクセシビリティ**: `prefers-reduced-motion`を尊重

**使用すべき場面**:
- 状態変化のフィードバック（ボタン押下、チェック完了）
- 要素の出現/消失（モーダル、トースト）
- ローディング/進捗表示

**使用を避ける場面**:
- 単なる装飾目的
- ユーザーの操作を遅延させるもの
- 長時間のアニメーション（1秒以上）

### 6.1 トランジション

| 要素 | 効果 | 時間 | イージング |
|------|------|------|-----------|
| ボタンホバー | 背景色・scale変化 | 150ms | ease |
| カードホバー | シャドウ拡大 | 200ms | ease |
| ページ遷移 | フェードイン | 300ms | ease-out |
| モーダル | スライドアップ + フェード | 250ms | ease-out |
| Accordion展開 | 高さアニメーション | 200ms | ease-in-out |
| トースト | スライドイン（右から） | 200ms | ease-out |

**CSS変数定義**:
```css
:root {
  --transition-fast: 150ms ease;
  --transition-normal: 200ms ease;
  --transition-slow: 300ms ease-out;
}
```

### 6.2 マイクロインタラクション

| 場面 | 効果 | 詳細 |
|------|------|------|
| **録音開始** | パルスアニメーション | 赤丸が1.2倍に拡大→縮小を繰り返し |
| **録音中** | 波形アニメーション | 3本のバーが高さを変える |
| **メッセージ到着** | スライドイン | 下からフェードイン、200ms |
| **約束完了** | チェックアニメーション | パスが描かれる + 軽いバウンス |
| **トーク終了** | 円形プログレス | 完了後にチェックマーク表示 |
| **エラー発生** | シェイク | 左右に2回揺れる、150ms |

### 6.3 ローディング状態

| 場面 | 表示 |
|------|------|
| ページ読み込み | スケルトンスクリーン |
| ボタン処理中 | ボタン内スピナー + disabled |
| データ取得中 | インラインスピナー |
| STT処理中 | パルスドット（...） |

**実装例**:
```css
/* パルスアニメーション（録音中） */
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.2); opacity: 0.8; }
}
.recording-indicator {
  animation: pulse 1.5s ease-in-out infinite;
}

/* Reduced motion対応 */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
